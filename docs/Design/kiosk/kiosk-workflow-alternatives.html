Kiosk Workflow - Refined Alternatives

Inspired by Société Générale's restrained, prestigious design language

Alternative 1: "Subtle Elegance"

Service colors are used sparingly as thin accent lines and badges. Primary CTA uses SG Red consistently. Clean white surfaces with generous breathing room. The focus is on typography and subtle color cues rather than bold blocks.

Design Principles

  • Service color as thin accent line only
  • White/light gray backgrounds throughout
  • SG Red (#E9041E) for primary actions
  • Subtle shadows, no heavy elevation

Color Application

  • Screen 1: Outlined cards with colored left border
  • Screen 2: Small colored badge, red CTA
  • Screen 3: Light tint background, colored ticket number

Alternative 2: "Corporate Refinement"

Black and white foundation with strategic red accents. Service differentiation through icons and typography rather than color blocks. Buttons use outlined style with subtle fills. The prestige comes from restraint and precision.

Design Principles

  • Primarily black, white, and gray
  • Red (#E9041E) only for key actions
  • Outlined buttons for secondary actions
  • Service icons in black, context through labels

Color Application

  • Screen 1: White cards, black icons, thin borders
  • Screen 2: Red primary CTA, outlined skip button
  • Screen 3: White background, subtle service tint

Alternative 3: "Prestige Minimal"

Maximum restraint. Almost entirely monochromatic until the final ticket reveal. Color is earned through the workflow - the ticket confirmation screen celebrates with a subtle color wash. Ultra-clean typography carries the experience.

Design Principles

  • Screens 1-2: Pure black and white
  • Color appears only on final screen
  • Typography-driven hierarchy
  • No colored icons until ticket

Color Application

  • Screen 1: Black icons on white, no color
  • Screen 2: Black CTA button, monochrome
  • Screen 3: Service color celebration
