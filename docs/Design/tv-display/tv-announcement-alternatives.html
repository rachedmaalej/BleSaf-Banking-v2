recommend Design Recommendations

Primary Recommendation: Classic Bottom Banner (#1)
Keep the current bottom banner approach but enhance with:
  • Smoother cubic-bezier easing
  • Visual progress bar for countdown
  • Larger icon for TV visibility
  • Slightly bigger text (20px minimum)
For Critical Alerts: Floating Card (#3)
Use the centered floating card for truly critical announcements (emergencies, closures). The backdrop blur ensures the message is unmissable. Reserve for urgent priority only.
Animation Best Practices
  • Use cubic-bezier(0.4, 0, 0.2, 1) for natural motion
  • Keep durations 300-500ms for entrance
  • Add subtle pulse for urgent messages
  • Always include progress indicator
Typography for TV
  • Minimum 18px font size, prefer 20-24px
  • Use font-weight 500-600 for messages
  • Ensure 7:1 contrast ratio
  • Max 2 lines for quick readability

Implementation Suggestion

Consider implementing a tiered approach: Use the enhanced Bottom Banner (#1) as the default for all normal announcements, and automatically switch to the Floating Card (#3) for urgent priority messages. This provides the best balance between non-intrusiveness for routine updates and maximum attention capture for critical alerts.
