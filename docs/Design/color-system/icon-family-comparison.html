Verdict

After evaluating all five families against BléSaf's corporate banking aesthetic, multi-context sizing needs (10px inline labels to 80px kiosk icons), and the SG brand guidelines, the recommendation is clear.

Material Symbols is already deeply integrated (253 uses across 42 files), has the best banking-specific icon coverage, and offers a unique variable font architecture that no other library matches. The only action needed is migrating the 9 Heroicons in AppLayout.tsx to complete a 100% unified system.

verified Material Symbols Outlined

Primary icon family — keep & complete migration
  • Already 253 uses across 42 files
  • Variable font: weight, fill, grade, optical size
  • Best banking icon coverage (local_atm, payments, receipt_long)
  • Scales perfectly from 10px to 80px
  • Zero migration cost

Phosphor Light

Runner-up — ideal complement for DoctorQ variant
  • Light weight pairs with Inter 300 display typography
  • Softer aesthetic for medical/healthcare contexts
  • Duotone variant adds depth without filling
