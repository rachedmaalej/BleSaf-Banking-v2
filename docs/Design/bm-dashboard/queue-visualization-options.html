Recommendation

Option A (Silhouette Row) provides the best balance of readability, intuitive understanding, and modern aesthetics. Each person icon clearly represents one customer, making it immediately understandable for all ages and languages. The "+N" overflow handling keeps the display clean even with large queues.

Alternative: Option F (Minimalist) if maximum distance readability is the priority and visual engagement is secondary.
