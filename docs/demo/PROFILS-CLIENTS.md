> Chaque personnage illustre une fonctionnalite cle de BléSaf.

| | |
|---|---|
| **Age** | 28 ans |
| **Metier** | Enseignante au lycee de Carthage |
| **Service** | Retrait d'especes (A) |
| **Priorite** | Normale |
| **Check-in** | Borne kiosk |
| **Telephone** | +216 22 456 789 — SMS |

**Histoire :** Amira arrive tot le matin pour retirer son salaire avant les cours. Elle est la premiere a prendre un ticket.

**Ce que le presentateur dit :**
